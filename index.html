#MuchProgramming {aka: Googling}

DraKodeus is an educational project to provide well commented and easy to read repositories for web dev learning. The projects show a progression of starting with pure HTML, CSS, and Javascript to the eventual introduction of other tools such as React, JQuery, etc.

Web development is more vocabulary than it is difficult. The repositories are meant to help young developers learn various skills such as utilizing trending vocabulary as troubleshoot and develop.
